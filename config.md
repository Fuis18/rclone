# Configurar rclone

## Instalarlo

https://rclone.org/downloads/

https://downloads.rclone.org/v1.72.0/rclone-v1.72.0-windows-amd64.zip

## Crear Path

Mover los archivos y carpetas a C:/rclone

```sh
cd rclone
```

## Crear cloud

```sh
rclone config
n
name> Cloudfuis
22
1 // Full access
```

## Ver Cloud
```sh
rclone lsd Cloudfuis
```
